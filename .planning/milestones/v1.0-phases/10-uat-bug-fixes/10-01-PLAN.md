**Bug 3 — Unit duplication:**
At ~line 131, the quantity column renders:
```typescript
`${formatQuantity(record.quantity)} ${record.unit || '米'}`
```
But `formatQuantity()` already appends the unit. Fix by either:
- Passing unit to formatQuantity and not appending it again, OR
- Using formatNumber instead of formatQuantity and appending unit once

Check `formatQuantity` signature in format.ts to determine the correct approach.

**Bug 4 — NaN purchase price:**
At ~lines 147-149, purchasePrice renders via AmountDisplay but shows ¥NaN when the value is null/undefined.
Add a null guard:
```typescript
render: (_, record) =>
  record.purchasePrice != null
    ? <AmountDisplay value={record.purchasePrice} suffix={`/${record.unit || '米'}`} />
    : '-',
```
