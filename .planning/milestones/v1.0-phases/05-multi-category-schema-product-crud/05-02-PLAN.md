**SubCategory to CodePrefix mapping** (private helper):
```typescript
private getCodePrefix(subCategory: string): CodePrefix {
  const map: Record<string, CodePrefix> = {
    [ProductSubCategory.IRON_FRAME]: CodePrefix.IRON_FRAME,
    [ProductSubCategory.MOTOR]: CodePrefix.MOTOR,
    [ProductSubCategory.MATTRESS]: CodePrefix.MATTRESS,
    [ProductSubCategory.ACCESSORY]: CodePrefix.ACCESSORY,
  };
  const prefix = map[subCategory];
  if (!prefix) throw new BadRequestException(`Unknown subCategory: ${subCategory}`);
  return prefix;
}
```

**Product CRUD methods** (follow FabricService pattern):
- `create(dto)`: generate productCode via CodeGeneratorService using subCategory prefix, $transaction with duplicate check, return created product

- `update(id, dto)`: $transaction, NotFoundException if not found, update fields

- `removeSupplier(productId, supplierId)`: find by @@unique, delete
