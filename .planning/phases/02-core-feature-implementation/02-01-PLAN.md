       - Add constructor dependencies: inject `RedisService` (from CommonModule, already global) and `CodeGeneratorService` (already injected)
       - Import `OrderItemStatus`, `calculateAggregateStatus` from `../order/enums/order-status.enum`
       - Import `ServiceUnavailableException`, `ConflictException` from `@nestjs/common`
       - Import `ConvertQuotesToOrderDto` from `./dto`
       - Import `Order` from `@prisma/client`

       Method signature: `async batchConvertToOrder(dto: ConvertQuotesToOrderDto): Promise<Order>`

       Implementation:
       a. Check Redis availability — if `!this.redisService.isAvailable()`, throw `new ServiceUnavailableException('Redis is required for conversion — concurrent protection unavailable')`
       b. Sort quote IDs: `const sortedIds = [...dto.quoteIds].sort((a, b) => a - b)` — prevents deadlocks
       c. Acquire locks for all IDs: iterate sortedIds, call `this.redisService.acquireLock('quote:convert:' + id, 30)`. If any lock fails, release already-acquired locks and throw `new ConflictException('Quote ' + id + ' is being converted by another request')`
       d. Wrap remaining logic in try/finally, finally releases all locks via `this.redisService.releaseLock('quote:convert:' + id)` for each sortedId
       e. Inside try, run `this.prisma.$transaction(async (tx) => { ... })`:
          - Fetch all quotes: `tx.quote.findMany({ where: { id: { in: sortedIds } }, include: { fabric: { select: { id: true } } } })`
          - Validate: all quotes found (throw NotFoundException listing missing IDs), all have status 'active' (throw BadRequestException listing non-active IDs with their statuses), none expired (validUntil > now, throw BadRequestException listing expired IDs), all same customerId (throw BadRequestException if mixed)
          - Extract customerId from first quote
          - Look up suppliers: `tx.fabricSupplier.findMany({ where: { fabricId: { in: fabricIds } }, orderBy: { purchasePrice: 'asc' }, distinct: ['fabricId'] })` — build Map<fabricId, supplierId>
          - Generate order code: `this.codeGeneratorService.generateCode(CodePrefix.ORDER)`
          - Calculate totalAmount: `quotes.reduce((sum, q) => sum + Number(q.quantity) * Number(q.unitPrice), 0)`
          - Create order with items:
            ```typescript
            const order = await tx.order.create({
              data: {
                orderCode,
                customerId,
                status: OrderItemStatus.PENDING,
                totalAmount,
                customerPaid: 0,
                customerPayStatus: 'unpaid',
                items: {
                  create: quotes.map(q => ({
                    fabricId: q.fabricId,
                    supplierId: supplierMap.get(q.fabricId) ?? null,
                    quoteId: q.id,
                    quantity: q.quantity,
                    salePrice: q.unitPrice,
                    subtotal: Number(q.quantity) * Number(q.unitPrice),
                    status: OrderItemStatus.PENDING,
                  })),
                },
              },
              include: { items: true, customer: true },
            });
            ```
          - Create timeline entries:
            ```typescript
            await tx.orderTimeline.createMany({
              data: order.items.map(item => ({
                orderItemId: item.id,
                fromStatus: null,
                toStatus: OrderItemStatus.PENDING,
                remark: 'Converted from quote',
              })),
            });
            ```
          - Update all quotes to CONVERTED: `tx.quote.updateMany({ where: { id: { in: sortedIds } }, data: { status: QuoteStatus.CONVERTED } })`
          - Return order
